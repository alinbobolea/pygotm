pyGOTM Documentation
====================

Python + Taichi reimplementation of the General Ocean Turbulence Model (GOTM).
GPU-accelerated 1D ocean/lake turbulence for oceanographers, limnologists,
and aquaculture site assessors — no Fortran compiler required.
